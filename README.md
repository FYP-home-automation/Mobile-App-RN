# FYP_Home_Automation

## Make sure you have these dependencies installed:
1. Node.js
2. expo-cli

## Set up the project and run the project: 

1. Clone the app
```
  git clone https://github.com/Kevintirta/FYP_Home_Automation.git
```

2. Install dependencies
```
  yarn install
```

3. Run app
```
  expo start
```
